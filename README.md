# nodejs-template

## Startup

```ssh
npx degit Vehmloewff/nodejs-template nodejs-app
```
```ssh
cd nodejs-app
```
```ssh
npm i
```

## Running the tests

To run jest:

```ssh
npm test
```

or

```ssh
npm test:watch
```

## Linting

This template uses a combination of `prettier` and `eslint`.

```ssh
npm run lint
```

or

```ssh
npm run lint:test
```

To have `eslint` and `prettier` ignore files, just add to the `.eslintignore` and `.prettierignore` files.
